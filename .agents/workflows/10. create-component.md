🚀 Create Atomic Component Workflow

Workflow này tự động hóa quy trình scaffolding, đăng ký, và xác minh một component mới trong Design System.

**Reference:** `ARCHITECTURE.md` § CSS Architecture, § JavaScript Module System, § Component Hierarchy, `/module-creation`, `/linting-gates`

---

## 📥 Input Requirements

Trước khi tạo component, xác định:

- **Name**: Tên component (kebab-case, ví dụ: `button`, `search-input`)
- **Level**: `atoms`, `molecules`, hoặc `organisms`
- **HasLogic**: `true` (cần JS) hoặc `false` (chỉ CSS)
- **Dependencies**: Các component khác được sử dụng (nếu có)

---

## 🛠️ Thực thi từng bước (Step-by-Step Execution)

### Bước 1: Khởi tạo File CSS (CSS Scaffolding)

**Đường dẫn:** `renderer/css/design-system/<level>/<name>.css`

**Cấu trúc chuẩn:**

```css
/* ── Design System: [Name] ([Level]) ── */

.ds-[name] {
  /* Layout */
  display: flex;
  align-items: center;
  justify-content: center;

  /* Sizing */
  width: auto;
  min-height: var(--ds-space-lg);
  padding: var(--ds-space-sm) var(--ds-space-md);

  /* Styling (ALWAYS use tokens) */
  background: var(--ds-bg-base);
  color: var(--ds-text-primary);
  border: 1px solid var(--ds-border-subtle);
  border-radius: var(--ds-radius-md);

  /* Transitions */
  transition: all var(--ds-transition-smooth);
}

/* Variants (if needed) */
.ds-[name].ds-[name]--primary {
  --_bg: var(--ds-accent);
  --_color: var(--ds-text-on-accent);
}

.ds-[name]:hover {
  background: var(--ds-bg-hover);
}

.ds-[name]:focus-visible {
  outline: 2px solid var(--ds-accent);
  outline-offset: 2px;
}
```

**Quy tắc:**
- Bắt buộc sử dụng CSS tokens (không hardcode màu/spacing)
- Sử dụng `var(--ds-...)` cho tất cả giá trị
- Sử dụng `--_varname` pattern cho local variants
- Bao gồm transitions + focus/hover states

### Bước 2: Khởi tạo File JS (JS Scaffolding)

**Chỉ khi HasLogic = true**

**Đường dẫn:** `renderer/js/components/<level>/<name>.js`

**Cấu trúc IIFE bắt buộc:**

```javascript
/**
 * [Name] Component ([Level])
 * Purpose: [Short description]
 * Dependencies: [List if any]
 */
const [PascalName] = (() => {
  'use strict';

  // ============================================
  // Private State
  // ============================================
  const SELECTORS = {
    root: '.ds-[name]',
    trigger: '.ds-[name]-trigger',
  };

  let _state = {
    // private state
  };

  // ============================================
  // Private Functions
  // ============================================
  function _initElement(element) {
    // Setup logic per element
  }

  function _attachListeners(element) {
    // Event listeners
  }

  // ============================================
  // Public API
  // ============================================
  return {
    init: function() {
      const elements = document.querySelectorAll(SELECTORS.root);
      elements.forEach(_initElement);
    },
    destroy: function() {
      // Cleanup if needed
    },
  };
})();

// Explicit export to global scope
window.[PascalName] = [PascalName];
```

**Quy tắc:**
- Bắt buộc IIFE pattern (no global state)
- Private functions với `_` prefix
- `init()` là entry point bắt buộc
- Explicit `window.*` export bắt buộc

### Bước 3: Đăng ký CSS (Register CSS)

**File:** `renderer/css/design-system.css`

Thêm import vào cuối file (bên dưới level tương ứng):

```css
/* Atoms */
@import 'design-system/atoms/button.css';
@import 'design-system/atoms/[new-name].css';  /* ← NEW */

/* Molecules */
@import 'design-system/molecules/context-menu.css';
@import 'design-system/molecules/[new-name].css';  /* ← NEW */

/* Organisms */
@import 'design-system/organisms/tab-bar.css';
@import 'design-system/organisms/[new-name].js';  /* ← NEW */
```

### Bước 4: Đăng ký JavaScript (Register JS)

**File:** `renderer/index.html`

Thêm script vào đúng vị trí theo script pipeline order:

```html
<!-- COMPONENTS (Atoms → Molecules → Organisms) -->
<script src="js/components/atoms/button.js" defer></script>
<script src="js/components/atoms/[new-name].js" defer></script>  <!-- ← NEW -->

<script src="js/components/molecules/search-component.js" defer></script>
<script src="js/components/molecules/[new-name].js" defer></script>  <!-- ← NEW -->

<script src="js/components/organisms/tab-bar.js" defer></script>
<script src="js/components/organisms/[new-name].js" defer></script>  <!-- ← NEW -->
```

**Quy tắc:**
- Tuân thủ script pipeline order (Core → Atoms → Molecules → Organisms → Services → Modules → App)
- Tất cả script phải có `defer` attribute
- Chèn vào đúng level (atoms/molecules/organisms)

### Bước 5: Kích hoạt trong App (Boot Component)

**File:** `renderer/js/core/app.js`

Tìm `DOMContentLoaded` hoặc `AppState.registerInitializer()` và thêm init call:

```javascript
DOMContentLoaded(() => {
  // ... existing init calls ...
  
  // Components
  IconActionButton.init?.();
  SwitchToggle.init?.();
  [PascalName].init?.();  // ← NEW
  
  // ... rest of app init ...
});
```

### Bước 6: Xác minh Linting (Verify Linting)

Chạy linting để đảm bảo không có lỗi:

```bash
npm run lint
```

**Nếu có lỗi CSS:**
- `color-no-invalid-hex` → Fix hex values
- `no-duplicate-selectors` → Merge duplicate blocks
- `function-calc-no-unspaced-operator` → Add spacing in calc()

**Nếu có lỗi JavaScript:**
- `no-unused-vars` → Prefix unused vars với `_`
- `eqeqeq` → Use `===` not `==`
- `no-undef` → Ensure `window.[PascalName] = [PascalName]` is present

### Bước 7: Cập nhật ARCHITECTURE.md (Document)

**File:** `ARCHITECTURE.md` § JavaScript Module System

Thêm entry cho component mới:

```markdown
## Component: [Name] ([Level])
- **Path**: `renderer/js/components/[level]/[name].js`
- **CSS**: `renderer/css/design-system/[level]/[name].css`
- **Type**: [Atom/Molecule/Organism]
- **Purpose**: [1-line description]
- **Public API**:
  - `init()` — Initialize component
  - `destroy()` — Cleanup
- **Dependencies**: [List if any]
```

## ⚠️ Nguyên tắc quan trọng

1. **`ds-` prefix bắt buộc**
   - Tất cả class phải bắt đầu với `ds-`
   - Tránh xung đột với legacy CSS

2. **Tokens không hardcode**
   - ❌ `color: #ff9800;`
   - ✅ `color: var(--ds-accent);`

3. **IIFE + window.* exports**
   - Tất cả JS components phải IIFE pattern
   - Explicit `window.*` export bắt buộc
   - Không global state outside IIFE

4. **Script pipeline order**
   - Tuân thủ Atoms → Molecules → Organisms
   - Dependencies load trước khi được sử dụng
   - Core modules load đầu tiên

5. **Linting gates**
   - `npm run lint` phải 0 errors trước khi commit
   - Không bao giờ bypass hoặc ignore linting
